HDF5R plotting tool.
Version: 0.99
Date: June 26, 2010
Author: Ryan Seal

Introduction:

This tool is designed for use with the HDF5 for radar header system. The
software provides 1 second RTI images for HDF5R radar data.

Dependencies:

1. Python
2. Matplotlib with the FLTK backend.
